# 📌 Linux shell 工具书

## 📝 排查日志

针对 后端日志系统。shell 上的日志排查就是对一堆 log 文本文件操作

⚙️ grep

查找包含特定关键字的行

- `grep "error" app.log` 查找 log 文件匹配 error 的内容
- `grep -i "error" app.log` 忽略大小写匹配
- `grep -n "example" file.txt` 显示行号
- `grep -r "example" .` 递归搜索当前目录及其子目录中的所有文件，查找包含“example”的行
- `grep -C 2 "example" file.txt`  查找包含“example”的行，并显示其前后两行的内容：
- `grep -w "example" file.txt` 只查找作为单独单词出现的“example”，而不是作为其他单词的一部分
