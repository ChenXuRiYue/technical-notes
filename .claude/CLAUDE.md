# 技术笔记知识库

个人技术笔记仓库，用于记录和整理各类技术知识。

## 目录结构

- 人工智能/ — AI 相关笔记（大模型、Agent、工具、知识专项等）
- 全栈工程/ — 前后端工程笔记（React、Tauri 等）
- 后端工程/ — 后端专项（Spring、Python 等）
- 语言基础/ — 编程语言基础笔记
- 专业基础/ — 计算机专业基础知识
- 开源项目/ — 开源项目经验与文档
- 项目学习/ — 项目实战学习记录
- 代码视觉/ — 代码可视化相关
- 竞技算法/ — 算法与竞赛笔记
- 开发环境/ — 开发环境配置与工具
- 个性配置/ — 个人工具与编辑器配置
- 素质体系/ — 综合素质相关

## 素质体系

素质体系用于记录工程素养和软技能，包含以下子目录：

- 技术方案/ — 技术方案编写、通用性能力、接口设计
- 信息素养/ — 文档能力（架构图、用例图、优雅文档写作）、文档阅读（阅读技巧与语境积累）
- 团队协作/ — 代码管理、协作模式、个人职能
- 文档模版/ — 各类文档模版（见下文）
- 整洁代码/ — 代码整洁之道与 CRUD 案例库

## 文档模版

仓库中预置了以下文档模版，创建新笔记时应参考对应模版：

### 技术文档 — 工具书模版
路径：`素质体系/文档模版/技术文档/工具书.md`
结构：标题 → 按功能分节（含使用场景、经验表格、引用链接）

### 技术文档 — 技术学习模版
路径：`素质体系/文档模版/技术文档/技术学习.md`
结构：标题 → 基础谈资 → 按 topic 分节 → 生长思考 → 反复绊脚

### 工作复盘模版
路径：`素质体系/文档模版/工作复盘/REDEME.md`
原则：不出现具体业务和人名，抽象为通用场景

### 目标规划模版
路径：`素质体系/文档模版/目标规划/plan.md`
结构：背景 → 规划 → 补充，遵循 SMART + OKR 原则

### 竞赛算法 — 周赛补题模版
路径：`素质体系/文档模版/竞赛算法/lc周赛补题.md`
结构：竞赛标题 → 排名/过题数/算法标签 → 复盘 → 补题

### 竞赛算法 — 算法学习模版
路径：`素质体系/文档模版/竞赛算法/算法模版.md`
结构：背景 → 应用 → 原理 → 模版代码 → 发散

### 通用规范
路径：`素质体系/文档模版/通用规范.md`
内容：主题颜色、标题格式、LaTeX 公式等排版规范

## 协作须知

- 作者常用语音转文字输入，文本中可能出现错别字或谐音字，遇到不通顺的词时优先从谐音字和上下文角度猜测原意

## 规范

- 使用中文撰写
- 文件格式为 Markdown
- 保持已有目录结构，新内容归入合适的分类
- 新建技术笔记时参考 `素质体系/文档模版/` 下对应的模版
- 追加经验内容到已有文档时，优先保留原有文档的风格和结构。如无法兼容，再向用户提出
