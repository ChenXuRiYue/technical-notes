# 📝 快捷键 — mac

## 🧑‍💻 应用

**⚙️ 打开终端**

1. Cmd + space. 键入 terminate 或 终端
   - 注意可能会跳转到已有终端上
2. Cmd + shift + T. 聚焦系统的应用键入该键，会创建一个新的窗口

🪏 **经验**

| 场景                     | 使用 |
| ------------------------ | ---- |
| 打开终端，调用大模型界面 | 1    |
|                          |      |


---

**⚙️ 打开应用**

1. **Cmd + 空格 键入 application name**

- 应用已打开，则启动应用
- 应用已打开，有多个界面。跳转到最近使用的界面

2. macOs 将会打开 Spotlight 搜索。
   Spotlight 是 macOS 的全局搜索和启动器，有以下功能：

   - 搜索文件、应用程序、联系人、邮件

   - 启动应用程序

   - 进行单位换算、数学信息

   - 执行快捷操作

🪏 **经验**

| 场景         | 使用 |
| ------------ | ---- |
| 切到某个页面 | 1    |
|              |      |

## 🗂️ Document

**⚙️ 显示隐藏文件夹**

Cmd + Shift + . (句点)

🪏 **经验**

| 场景                                            | 使用 | 频率 |
| ----------------------------------------------- | ---- | ---- |
| 查看 .m2 maven 本地库<br />资源存放文件夹被隐藏 | 1    |      |
| 查看 git 仓库隐藏文件夹                         | 1    |      |

## 🌐 浏览器

**⚙️ 开发者模式**

`Option + Command + I`

🪏 **经验**

| 场景                       | 使用 |
| -------------------------- | ---- |
| 浏览器开发者模式找登录凭证 | 1    |

**⚙️ 刷新当前页面**

普通刷新：`Command + R`

强制刷新：`Command + Shift + R`

🪏 **经验**

| 场景                                             | 使用 |
| ------------------------------------------------ | ---- |
| 重新请求刷新页面。<br />避免资源过旧或刷新不出来 | 1    |

**⚙️ 新建标签页**

`Command + T`

**⚙️ 关闭标签页**

`Command + W`

**⚙️ 重新打开最近关闭的标签页**

`Command + Shift + T`
