# 📝 Mac 实用工具

Mac 平台实用工具积累

## 🔖 Snipaste — 截图贴图工具

⚙️ **安装**

```bash
brew install --cask snipaste
```

也可在 [官网](https://www.snipaste.com) 或 Mac App Store 下载。

⚙️ **核心功能与快捷键**

| 功能 | 快捷键 / 操作 |
| ---- | ---- |
| 截图 | `F1`（默认）或自定义 |
| 截图后贴图（悬浮） | 截图区域选好后按 `Space`，图片钉在屏幕上 |
| 关闭贴图 | 右键贴图 → 关闭，或 `Esc` |
| 缩放贴图 | 滚轮 / 触控板双指缩放 |
| 移动贴图 | 拖动贴图 |
| 标注 | 截图后使用工具栏添加箭头、文字、马赛克等 |
| 多贴图 | 可同时钉多张截图在屏幕上，互不干扰 |

🪏 **经验**

| 场景 | 使用 |
| ---- | ---- |
| 对照参考 | 把图片钉在屏幕角落，边看边操作 |
| 多图对比 | 钉两张图在屏幕上，滚轮缩放，左右对比 |
| 快速分享 | 截图 → 标注 → `Cmd+C` 复制 → 粘贴到聊天窗口 |
