#### ⚙️ 跳转到变量的定义

⇧⌘B

🪏 **经验**

对于普通的变量成员，如果其为类对象，将会跳转到对应的定义
在认识这个快捷键前，我的工作流

1. 点击类变量
2. 跳转到类变量的声明定义处
3. 点击类
4. 跳转到类定义

该快捷键下，可以直接跳转到类定义节省了两步

| 场景                     | 使用 |
| ------------------------ | ---- |
| 跳转到当前 bean 的定义处 | 2    |
