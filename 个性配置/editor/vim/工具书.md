# 📌 Vim 命令清单

## 🔖 模式切换

| 快捷键 | 说明 |
|--------|------|
| `i` | 进入插入模式（光标前） |
| `a` | 进入插入模式（光标后） |
| `I` | 行首进入插入模式 |
| `A` | 行尾进入插入模式 |
| `v` | 可视模式（字符选择） |
| `V` | 可视模式（行选择） |
| `Esc` | 返回普通模式 |

## 🔖 光标移动

| 快捷键 | 说明 |
|--------|------|
| `h/j/k/l` | 左/下/上/右 |
| `w` | 下一个单词开头 |
| `b` | 上一个单词开头 |
| `e` | 当前单词结尾 |
| `0` | 行首 |
| `^` | 行首第一个非空字符 |
| `$` | 行尾 |
| `gg` | 文件开头 |
| `G` | 文件结尾 |
| `:n` | 跳转到第n行 |
| `Ctrl+f` | 向下翻页 |
| `Ctrl+b` | 向上翻页 |
| `Ctrl+d` | 向下翻半页 |
| `Ctrl+u` | 向上翻半页 |

## 🔖 编辑操作

| 快捷键 | 说明 |
|--------|------|
| `x` | 删除光标所在字符 |
| `X` | 删除光标前一个字符 |
| `dd` | 删除当前行 |
| `dw` | 删除到下一个单词开头 |
| `d$` | 删除到行尾 |
| `d0` | 删除到行首 |
| `yy` | 复制当前行 |
| `yw` | 复制到下一个单词开头 |
| `y$` | 复制到行尾 |
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |
| `u` | 撤销 |
| `Ctrl+r` | 重做 |
| `.` | 重复上一个命令 |

## 🔖 搜索替换

| 快捷键 | 说明 |
|--------|------|
| `/pattern` | 向下搜索 |
| `?pattern` | 向上搜索 |
| `n` | 下一个匹配项 |
| `N` | 上一个匹配项 |
| `:s/old/new/` | 替换当前行第一个匹配 |
| `:s/old/new/g` | 替换当前行所有匹配 |
| `:%s/old/new/g` | 替换文件中所有匹配 |

## 🔖 文件操作

| 快捷键 | 说明 |
|--------|------|
| `:w` | 保存文件 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 强制退出不保存 |
| `:e filename` | 打开文件 |
| `:r filename` | 读取文件内容到光标后 |
| `:w filename` | 另存为 |

## 🔖 窗口操作

| 快捷键 | 说明 |
|--------|------|
| `:sp filename` | 水平分割窗口 |
| `:vsp filename` | 垂直分割窗口 |
| `Ctrl+w h` | 切换到左侧窗口 |
| `Ctrl+w j` | 切换到下方窗口 |
| `Ctrl+w k` | 切换到上方窗口 |
| `Ctrl+w l` | 切换到右侧窗口 |
| `Ctrl+w q` | 关闭当前窗口 |
| `Ctrl+w =` | 等分窗口大小 |

## 🔖 文本对象

| 快捷键 | 说明 |
|--------|------|
| `ci"` | 修改引号内的内容 |
| `ci'` | 修改单引号内的内容 |
| `ci(` | 修改括号内的内容 |
| `ci[` | 修改方括号内的内容 |
| `di"` | 删除引号内的内容 |
| `di(` | 删除括号内的内容 |
| `da"` | 删除引号及其内容 |
| `da(` | 删除括号及其内容 |

## 🔖 标记跳转

| 快捷键 | 说明 |
|--------|------|
| `m{a-z}` | 设置标记（a-z） |
| `` {a-z}` | 跳转到标记位置 |
| `''` | 跳转到上一个位置 |
| `` ``` | 跳转到上一个跳转位置 |

## 🔖 宏录制

| 快捷键 | 说明 |
|--------|------|
| `q{a-z}` | 开始录制宏到寄存器a-z |
| `q` | 停止录制 |
| `@{a-z}` | 执行寄存器中的宏 |
| `@@` | 重复执行上一个宏 |

## 🔖 寄存器操作

| 快捷键 | 说明 |
|--------|------|
| `"{a-z}y` | 复制到指定寄存器 |
| `"{a-z}p` | 从指定寄存器粘贴 |
| `:reg` | 查看所有寄存器内容 |
| `"+y` | 复制到系统剪贴板 |
| `"+p` | 从系统剪贴板粘贴 |

## 🔖 高级操作

| 快捷键 | 说明 |
|--------|------|
| `zi` | 切换折叠 |
| `za` | 切换当前折叠 |
| `zc` | 关闭当前折叠 |
| `zo` | 打开当前折叠 |
| `zM` | 关闭所有折叠 |
| `zR` | 打开所有折叠 |
| `Ctrl+n` | 向下搜索补全 |
| `Ctrl+p` | 向上搜索补全 |
| `:tabnew` | 新建标签页 |
| `:tabclose` | 关闭当前标签页 |
| `:tabn` | 切换到下一个标签页 |
| `:tabp` | 切换到上一个标签页 |
| `gt` | 切换到下一个标签页 |
| `gT` | 切换到上一个标签页 |

## 🔖 自定义配置

- 使用 jj 代替 ESC

```vim
inoremap jj <esc>
```
