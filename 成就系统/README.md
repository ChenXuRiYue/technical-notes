# 🎮 游戏化

游戏日志与赛季归档存放于此。

## 目录结构

```
成就系统/
├── README.md          ← 本文件
├── GAMIFICATION.md    ← 游戏机制规则
├── AGENT_RULES.md     ← Agent 裁判行为规范
├── CHANGELOG.md       ← 版本变更日志
├── 垃圾桶.md          ← 放弃的任务（不计分，留痕）
├── 数据/              ← 结构化存储，审计对账，未来迁移用
│   ├── exp.jsonl      ← 经验流水（只追加）
│   ├── coin.jsonl     ← 金币流水（收入+支出，只追加）
│   └── state.json     ← 当前状态快照
└── 回顾/
    ├── 周/
    ├── 月/
    ├── 季/
    ├── 年/
    └── 境界/          ← 境界突破留念（一生一次的人生刻度）
```

## 游戏日志范式

### 周回顾

文件名：`YYYY-WNN.md`（如 `2026-W23.md`）

```markdown
# 📅 2026-W23 周回顾

- 完成任务：N 个
- 获得经验：+X
- 获得金币：+X
- 连击：当前 N 连
- 高光任务：任务名（档位）

> 状态：📊 境界：X | 经验：X | 🌙 月进度 X/阈值
```

### 月回顾

文件名：`YYYY-MM.md`（如 `2026-06.md`）

```markdown
# ⛰️ 2026-06 月回顾

## 数据

- 完成任务：N 个
- 获得经验：+X
- 获得金币：+X
- 登山等级：🚩登顶
- 对比上月：↑/↓/→

## 周一览

| 周 | 任务数 | 经验 |
|----|--------|------|
| W23 | 5 | 21 |
| W24 | 3 | 11 |
| ... | | |

## 高光

- 最高经验任务：任务名（档位）
- 解锁成就：成就名（如有）
```

### 季回顾

文件名：`YYYY-QN.md`（如 `2026-Q2.md`）

```markdown
# 🌊 2026-Q2 季回顾

## 数据

- 完成任务：N 个
- 获得经验：+X
- 海域等级：🌀远洋
- 最高连击：N

## 月一览

| 月 | 登山 | 经验 |
|----|------|------|
| 4月 | 🚩登顶 | 92 |
| 5月 | 🏔️雪线 | 75 |
| 6月 | 🥾攀登 | 45 |

## 里程碑

- [项目名] 完结 / 推进至 XX%
- Boss 阶段推进（如有）
```

### 年回顾

文件名：`YYYY.md`（如 `2026.md`）

```markdown
# 🌍 2026 年回顾

## 数据

- 完成任务：N 个
- 获得经验：+X
- 大陆等级：🌍大陆
- 经验总量：+X
- 境界变化：🌠 微光 → 🌟 星火（如有突破）

## 季一览

| 季度 | 海域 | 经验 |
|------|------|------|
| Q1 | 🌊近海 | 140 |
| Q2 | 🐉深渊 | 310 |
| Q3 | 🌀远洋 | 270 |
| Q4 | 🏖️港湾 | 400 |

## 成就解锁

- 🏆 成就名 (日期)
- ...

## Boss 进度

- 🐉 GitTributary：30% → 45%

## 年度高光 Top 3

1. 任务名（💀 +25）
2. 任务名（🔴 +15）
3. 任务名（🔴 +15）
```
