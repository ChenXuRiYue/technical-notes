# 🎮 游戏化机制

> 任务不是负担，是经验值。

## 积分体系

### 两种货币

- **经验（成长货币）**：只增不减，永不消耗。驱动赛季进度，是境界试炼的底分。代表"我成长到哪了"。
- **金币（犒劳货币）**：可消费。只用于奖励商店，犒劳现实中的自己。代表"我能给自己多少休息/享乐"。

> 二者的取舍是真实的：金币攒着不花，还是兑换一杯咖啡放松一下？这是工作-休息的平衡决策。

### 公式（统一）

```
经验 = 时间档位
金币 = 该笔经验 × 0.5（向下取整）
```

**所有经验来源**（任务、连击、成就、完结奖励）都按同一规则同步产出金币。

### 时间档位

完成任务时由用户自标（勾选时末尾加 emoji），时间是客观的、只有本人知道。

| 档位 | 名字 | 时间 | 经验 | 金币 |
|------|------|------|------|------|
| ⚡ | 顺手 | < 15min | 1 | 0 |
| 🟢 | 轻松 | 15min - 1h | 3 | 1 |
| 🟡 | 投入 | 1h - 4h | 7 | 3 |
| 🔴 | 硬仗 | 4h - 1天 | 15 | 7 |
| 💀 | 鏖战 | > 1天 | 25 | 12 |

只评价"投入时间"，不评价"价值"——客观、防作弊、零额外 token。忘标则 git 兜底按最低档 ⚡。

> 计分只奖励"投入"，不评价"价值"——这是为了客观与防作弊。**成就感由赛季、成就、挑战、传奇 Boss 等上层结构提供，不指望单任务计分。**

## 爆率系统（暴击）

给任务注入"老虎机"惊喜：爆率创建时定且可见（制造期待），摇奖在审计那刻才兑现。

### 价值评估模型（5 维，命中即记 1 分）

新增任务时由 Agent 按作者画像评估，偏保守，拿不准下调一档。

| 维度 | 命中条件 |
|------|---------|
| 🌐 体系 | 推进 GitTributary / 游戏系统 / 个人信息系统 / SKILL 化 |
| ⚔️ 攻坚 | 攻克全新领域 / 困难但正确 |
| 🛠️ 造物 | 产出可复用资产：方法论/框架/SKILL/prompt/模版/脚本（**纯描述笔记不算**） |
| 🩹 痛点 | 源于真实反复痛点：工作流摩擦/机械重复/AI 协作卡点/token 浪费（**非"以后可能有用"**） |
| ✨ 趣味 | 主观创造欲 / 情绪价值 |

### 爆率档（保守映射，标在待办上）

| 标记 | 爆率 | 命中数 |
|------|------|--------|
| 🎲 普通 | 5% | 0-1 |
| 🎯 有趣 | 15% | 2-3 |
| 🎰 头奖 | 30% | ≥4 **或** 命中 🌐体系 |

> 甜点区：把真实痛点做成 SKILL/自动化 → 同时命中 🌐+🛠️+🩹 → 直接 🎰。

### 摇奖（审计那刻才确定，可复算）

随机源用**审计时间戳哈希**，因为审计在 commit 之前、hash 尚不存在：

```
roll = hash(审计时间戳 + 任务名) % 100
roll < 爆率 → 暴击！本次任务基础经验 ×2
```

账本记 `rate / roll / crit`，任何人拿时间戳+任务名可复算验证，AI 无法作弊。

### 防膨胀

- 暴击倍率固定 **×2**，写死不可调（调即通胀）
- 只作用于**单任务基础经验**，不碰连击/成就/赛季基数
- 期望增益有界：全 🎰30% 也仅平均 +30% 基础经验

| 类型 | 例子 | 打分方式 | 额外机制 |

| Boss | 状态 | 说明 |
|------|------|------|
| 🐉 GitTributary | 🟡 进行中 | 贯穿一生的基础设施 |

## 生涯境界（星辰）

人生里程碑层，**与赛季、积分无关**。赛季是积分进度的自动盖章；境界是"我成为了不一样的人"的质变，靠**试炼 + 仪式**进阶。一生可能只走过三五重，跨度以年计。

| 境界 | 内核 | 突破试炼 | 跨度 |
|------|------|---------|------|
| 🔥 火种 | 初始状态 | 游戏初始化即达成（起点） | 此刻 |
| 🌠 微光 | 学会坚持 | 累计活跃 ≥180 天 **且** 累计完成 ≥100 任务 **且** 连续 ≥6 月无断更归零 | ~1 年（2026 目标）|
| 🌟 星火 | 思想凝结 | 完成 1 个思想结晶级造物（如 GitTributary）+ ≥15 个 🔴/💀 + ≥8 个造物 | 2-3 年 |
| ⭐ 恒星 | 待定 · 自成体系 | 走到星火再结晶。松散方向：能稳定自发光、不依赖外部驱动地持续产出 | 待定 |
| 🌌 银河 | 待定 · 留下印记 | 完全开放，不绑定职业身份——技术若成爱好，亦可是"在热爱的事上留印记" | 待定 |

> 恒星、银河刻意留白：三年后是否仍为全职工程师未可知，等走到星火再定义远方。当前只聚焦 火种 → 微光 → 星火。

### 突破仪式

试炼达成时，系统只提示「你已具备突破【X】的条件」，**不自动进阶**。由你选一个郑重时刻**亲自确认**，写一句留念，存入 `成就系统/回顾/境界/`。突破是主动的仪式，不是被动达标。

## 月度挑战

提供"主动追目标"的牵引力，对抗待办的被动性。每月一条，月初由裁判设定，存于 `state.season.month_challenge`。

挑战示例（裁判从中选一，或按当月状态生成）：

| 挑战 | 达成 | 奖励 |
|------|------|------|
| 🎖️ 攻坚月 | 本月完成 2 个 🔴 及以上任务 | +15 + 隐藏徽章 |
| 🎖️ 广度月 | 本月覆盖 4 个不同领域标签 | +15 + 隐藏徽章 |
| 🎖️ 沉淀月 | 本月沉淀 5 篇笔记 | +15 + 隐藏徽章 |
| 🎖️ 勤勉月 | 本月活跃天数 ≥15 天 | +15 + 隐藏徽章 |

未完成不惩罚，月末清空换新。

## 成就系统

所有成就均可从 `state.json` 廉价判定，无需扫全库。已解锁记入 `achievements[]`，不重复触发。

### 明置成就（里程碑，展示进度）

| 成就 | 条件 | 依据字段 | 奖励 |
|------|------|---------|------|
| 🏆 初出茅庐 | 完成第 1 个任务 | tasks_completed≥1 | +5 |
| 🔥 连击达人 | 达成 3 连击 | combo≥3 | +10 |
| 💀 地狱征服者 | 完成首个 💀 任务 | 本次档位=💀 | +20 |
| 🗓️ 每日一更 | 连续 7 天有完成 | daily_streak≥7 | +15 |
| 📚 知识沉淀 | 沉淀 5 篇笔记 | notes_count≥5 | +15 |
| 🏅 百分里程碑 | 累计经验达 100 | exp_total≥100 | +10 |
| 📦 交付者 | 首次完结一个项目 | project_complete | +15 |
| 🌟 突破！ | 首次突破新境界 | 跨境界 | +25 |

### 隐藏成就（不展示条件，结算时揭晓）

判定只看本次 ledger 时间戳 + state，零额外成本。

| 成就 | 触发 | 奖励 |
|------|------|------|
| 🌅 破晓 | 06:00-09:00 完成任务（夜行者的稀有时段） | +10 |
| 🧹 洁癖 | 单次 commit 同时归档 ≥3 个任务 | +10 |
| 🎯 效率怪物 | 单日完成 ≥3 个任务 | +10 |
| 🐉 王者归来 | 断更 ≥30 天后回归并完成 💀 | +20 |
| 🌈 博学者 | 单周完成 3 个不同领域标签的任务 | +10 |

## 防膨胀规则

1. **积分不回溯**：规则改动只影响未来，已发放的分数永远不动
2. **等级只升不降**：经验只增不减
3. **赛季按时间轮转**：月/季/年固定周期，不因不满意重开
4. **单任务有天花板**：最高 25 经验，可预测可设计

## 赛季回顾

每个赛季结束时，裁判生成一份回顾：

### ⛰️ 月回顾（每月末）

- 本月完成任务数、获得经验/金币
- 登山等级盖章
- 与上月对比（↑/↓/→）
- 高光任务（耗时最长的那个）

### 🌊 季回顾（每季末）

- 本季三个月登山一览
- 海域等级盖章
- 本季完成项目/里程碑
- 连击最高纪录

### 🌍 年回顾（每年末）

- 四个季度海域一览
- 大陆等级盖章
- 年度数据总览（总任务数、总经验、境界变化）
- 年度成就解锁列表
- Boss 进度推进情况
- 年度高光时刻（经验最高的 3 个任务）
