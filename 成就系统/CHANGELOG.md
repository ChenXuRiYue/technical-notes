## v1.3 — 重构：成就系统

**日期**：2026-06-04

### 变更内容

**目录与命名**

- `游戏化/` → `成就系统/`、`TODO.md` → `任务面板.md`；全库引用同步，校正 23 处失效路径
- 补齐 CHANGELOG v1.1 / v1.2 缺失的版本头

**任务面板**

- 按爆率分区重排（🎰头奖 / 🎯有趣 / 🎲普通），任务行不再带符号，爆率由所在分区表达
- 新增「📥 待定区」：随手裸写入口；读面板时派 subagent 按 5 维评估自动归类进分区
- 新增「🎁 随机推荐」：教授荐题落点
- 新增 🗑️ 垃圾桶（`成就系统/垃圾桶.md`）：放弃任务剪切至此，不计分、不惩罚

**生涯境界（星辰）**

- 境界与积分解耦：砍掉 50 级积分制境界，改为 5 重「星辰」里程碑层（🔥火种/🌠微光/🌟星火/⭐恒星/🌌银河）
- 靠试炼+仪式进阶：达成后系统只提示，由作者亲自确认写留念，存入 `回顾/境界/`
- 梯度：微光=180天活跃+100任务+6月不断更（2026目标）；星火含 GitTributary 级造物；恒星、银河留白
- 赛季降为三层（月/季/年）；state 去掉 `level`/`next_level_at`，改 `realm`+`active_days_total`，顶栏改显境界

### 影响

- 纯结构调整，不动已发放积分。GitTributary 从"终极传说"回归为境界三的一个造物。

## v1.2 — 档位自标 + 爆率系统

**日期**：2026-06-04

### 变更内容

- **档位改自标**：时间档位由用户勾选时敲数字 1-5（git 推断降为兜底），解决"git 测不准时间"的根本缺陷
- **爆率系统**：新增 5 维价值评估模型（🌐体系/⚔️攻坚/🛠️造物/🩹痛点/✨趣味），映射 3 档爆率（🎲5%/🎯15%/🎰30%）；暴击经验 ×2
- **摇奖随机源**：审计时间戳哈希 `hash(ts+任务名)%100`，可复算可验证（因审计早于 commit，不能用 commit hash）
- **连击降档**：+1/+2/+3 → 2 连 +1、≥5 连 +2
- **隐藏成就调整**：删 🌃夜行者（作者是夜行者会滥发），加 🌅破晓、🧹洁癖
- **画像固化**：基于全库 279 次提交 + 内容分析，锚定造物=方法论/框架、痛点=工作流摩擦/token浪费

### 影响

- 暴击只作用单任务基础经验，倍率 ×2 写死；对历史积分无影响（不回溯）。

## v1.1 — 货币定位 + 结算卡 + 数据修复

**日期**：2026-06-04

### 变更内容

- **数据修复**：ledger 加 `genesis` 锚定记录（exp:10）；修正 level 1→3（exp=10 对应 Lv.3，纯投影修正非回溯）
- **货币定位钉死**：经验=成长货币（不可消费）｜金币=犒劳货币（可消费），取舍=工作-休息平衡
- **金币规则统一**：所有经验来源 × 0.5 产出金币；新增 `redeem` 消费记录类型
- **成就拆分**：明置成就（里程碑，可从 state 字段廉价判定）+ 隐藏成就（惊喜，结算时揭晓）
- **新增月度挑战**：每月一条，提供主动牵引力
- **结算卡**：每次审计输出开箱式结算卡，修复"行动-反馈"断裂的核心循环
- **新增计数器**：state 加 `daily_streak`/`last_active_date`/`notes_count`/`month_challenge`

### 影响

- 对已发放积分无影响（不回溯）。仅修正了 level 投影错误与补齐 genesis 锚点。

## v1.0 — Game Begin

**日期**：2026-06-03  
**提交**：Game Begin

### 确立的核心规则

- **积分公式**：经验 = 时间档位（⚡1/🟢3/🟡7/🔴15/💀25），金币 = 经验 × 0.5 向下取整
- **零主观判断**：裁判从 git 历史推断时间档位，不追问用户，不评价质量
- **四层赛季**：月（天气）/ 季（海域）/ 年（山峰）/ 生涯（50级境界），共享同一份积分
- **目标分类**：单任务直接打分 / 项目拆子任务+完结奖励 / 传奇Boss拆子任务+阶段勋章
- **防膨胀**：积分不回溯、等级只升不降、赛季按时间固定轮转
- **经验/金币分离**：经验升级用不消耗，金币奖励商店消费用
- **连击**：+1/+2/+3，7天断连归零不扣分
- **审计**：裁判从 git log 自主推断，不追问
- **数据存储**：ledger.jsonl 只追加不修改，state.json 为投影

### 初始状态

```json
{
  "level": 1,
  "realm": "新手村",
  "exp_total": 10,
  "coin_balance": 0,
  "combo": 2,
  "tasks_completed": 2
}
```

### 历史积分说明

v1.0 之前有 2 个任务按旧规则记了 10 分，直接继承为 10 经验，不回溯重算。
