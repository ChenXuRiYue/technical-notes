## ⚙️ latex积累

${\large {\color[RGB]{250, 157, 30}表格}} $

- **连字符 `-`**： 至少需要三个连字符来创建列的分隔线。连字符的数量可以多于三个，不影响显示。
- **冒号 `:`**： 在分隔线中添加冒号可以设置列的对齐方式。
- **竖线 `|`**： 用于分隔不同的单元格。

**两列**

```markdown
| 1| 2 |
|-------|-------|
|  |  |
|  |  |
```

**三列**

```markdown
| 1| 2 | 3 |
|-------|-------|-------|
|  |  | |
|  |  | |
```

**四列**

```markdown
| 1| 2 | 3 | 4 ｜
|-------|-------|-------|-------|
|  |  | | |
|  |  | | |
```
