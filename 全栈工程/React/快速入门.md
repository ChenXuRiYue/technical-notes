## 📄 显示数据

- JSX 将标签放到 JavaScript 中，而大括号支持让回到 JavaScript，从代码块中选取出变量展示给用户

  ```react
  return (
    <h1>

    </h1>
  );
  ```

  还可以将 JSX 属性 “转义到 JavaScript”

  ```react

  ```

## 📄 条件渲染

React 没有特殊的语法来编写条件语句，使用的就是普通的 JavaScript 代码。

**风格一：**

```react
let content;
if (isLoggedIn) {
  content = <AdminPanel />;
} else {
  content = <LoginForm />;
}
return (
  <div>
    {content}
  </div>
);
```

**风格二：**

```react
<div>
  {isLoggedIn ? (
    <AdminPanel />
  ) : (
    <LoginForm />
  )}
</div>
```

**风格三：**

```react
<div>
  {isLoggedIn && <AdminPanel />}
</div>
```



## 📄 渲染列表

数据来源：

```react
const products = [
  { title: 'Cabbage', id: 1 },
  { title: 'Garlic', id: 2 },
  { title: 'Apple', id: 3 },
];
```
在组件中，使用 map() 函数将这个数组转换为 <li> 标签构成的列表:
```react
const listItems = products.map(product =>
  <li key={product.id}>
    {product.title}
  </li>
);

return (
  <ul>{listItems}</ul>
);
```

## 📄 响应事件

```react
function MyButton() {
  function handleClick() {
    alert('You clicked me!');
  }

  return (
    <button onClick={handleClick}>
      点我
    </button>
  );
}
```

注意，`onClick={handleClick}` 的结尾没有小括号！不要 **调用** 事件处理函数：只需 **把函数传递给事件** 即可



## 📄 更新界面

DEMO：组件 “记住” 一些信息并展示出来，比如一个按钮被点击的次数。

- 引入 `useState`

  ```react
  import { useState } from 'react';
  
  ```

- 组件中声明一个 `state` 变量

  ```react
  function MyButton() {
    const [count, setCount] = useState(0);
    // ...
  ```

- 捕捉动作，并处理 state 变量

  ```react
  function MyButton() {
    const [count, setCount] = useState(0);
  
    function handleClick() {
      setCount(count + 1);
    }
  
    return (
      <button onClick={handleClick}>
        Clicked {count} times
      </button>
    );
  }
  ```

## 📄 组件中共享数据

多个组件之间共同一组数据的方法：

- 将数据在根组件中维护

  ```react
  export default function MyApp() {
    const [count, setCount] = useState(0);
  
    function handleClick() {
      setCount(count + 1);
    }
  
    return (
      <div>
        <h1>独立更新的计数器</h1>
        <MyButton />
        <MyButton />
      </div>
    );
  }
  
  function MyButton() {
    // ... 我们正在从这里移动代码...
  }
  ```

- 将 `MyApp` 中的点击事件处理函数以及 **state 一同向下传递到** 每个 `MyButton` 中

	```react
	export default function MyApp() {
  const [count, setCount] = useState(0);
	
  function handleClick() {
    setCount(count + 1);
  }
	
  return (
    <div>
      <h1>共同更新的计数器</h1>
      <MyButton count={count} onClick={handleClick} />
      <MyButton count={count} onClick={handleClick} />
    </div>
  );
	}
	```

- `MyApp` 组件包含了 `count` state 以及 `handleClick` 事件处理函数，并将它们作为 **prop 传递给** 了每个按钮。



## 🌳 生长思考

1. JSX 是什么？

   > **JSX**（全称：**JavaScript XML**）是一种语法扩展，它允许你在 JavaScript 代码中编写类似 HTML 的结构。
   >
   > JSX 不是 JavaScript 语法的一部分，而是一种“编译前的糖”
   >
   > JavaScript 引擎（如 V8、SpiderMonkey）**根本不认识 JSX**。
   > 但我们可以借助 **编译工具（如 Babel、TypeScript、SWC）**，在代码交给浏览器或 Node.js 执行之前，**先把 JSX 转成标准 JS**。
